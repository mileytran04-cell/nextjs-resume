# Next.js Resume Page

A single-page resume built with Next.js (App Router) and styled with Tailwind CSS.

## How to Run Locally
```bash
npm install
npm run dev
```

Visit http://localhost:3000 to view the page.

## Features

- [x] Next.js App Router project scaffolded with Tailwind CSS
- [x] Resume content in src/app/page.js using JSX
- [x] Semantic HTML elements (header, main, section, footer)
- [x] All styling done with Tailwind utility classes via className
- [x] Clean, readable layout at desktop viewport
- [x] No console errors

## Known Issues / Limitations

- Layout is optimized for desktop viewport only
- No mobile responsive breakpoints added yet
